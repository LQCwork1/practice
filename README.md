# Meeting Room Reservation System（会议室预约系统）

一个基于 C++ 控制台实现的简单会议室预约系统，用于练习面向对象编程与文件读写。

---

##  项目简介

该项目模拟企业内部会议室预约流程，实现用户登录、预约、查看、审核等功能。  
主要用于巩固 C++ 基础、类设计以及文件持久化操作。
> 本项目为学习练习项目，未进行复杂工程结构划分，所有源码文件位于同一目录。

---

##  功能实现

- 用户登录（员工 / 经理 / 管理员）
- 预约会议室
- 查看预约记录
- 取消预约
- 审核预约
- 添加账号

---

##  技术点

- C++ 面向对象编程（类的设计与封装）
- 多文件结构（.h / .cpp 分离）
- 文件读写（fstream 持久化数据）
- 简单的控制台交互逻辑

---

##  设计思路

- 将系统角色拆分为：
  - 员工类（Staff）
  - 经理类（Manager）
  - 管理员类 (Admin)
  - 会议室类（Room）

- 使用文本文件存储预约记录，实现简单持久化
- 通过菜单驱动控制流程（控制台交互）
- 不同身份拥有不同操作权限

---

##  运行方式（Visual Studio）

1. 使用 Visual Studio 创建空项目
2. 将所有 .cpp 和 .h 文件添加到项目中
3. 将所有 .txt 数据文件复制到项目的运行目录
4. 运行项目

>  注意：程序依赖 txt 文件存储数据，必须与可执行文件在同一目录，否则无法正常运行

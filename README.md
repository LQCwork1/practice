# Meeting Room Reservation System（会议室预约系统）

C++で開発したコンソールベースの会議室予約システムです。  
オブジェクト指向プログラミングとファイル入出力の練習を目的としています。

---

## 项目简介 / プロジェクト概要

该项目模拟企业内部会议室预约流程，实现用户登录、预约、查看、审核等功能。  
主要用于巩固 C++ 基础、类设计以及文件持久化操作。

本プロジェクトは、社内の会議室予約システムを簡易的に再現したものです。  
ユーザーログイン、予約、確認、承認などの機能を実装しています。

> 本项目为学习练习项目，未进行复杂工程结构划分，所有源码文件位于同一目录。  
> 学習目的のプロジェクトであり、複雑なプロジェクト構成は採用していません。

---

## 功能实现 / 主な機能

- 用户登录（员工 / 经理 / 管理员）  
- 预约会议室  
- 查看预约记录  
- 取消预约  
- 审核预约  
- 添加账号  

- ログイン機能（社員 / マネージャー / 管理者）  
- 会議室予約  
- 予約確認  
- キャンセル  
- 予約承認  
- アカウント追加  

---

## 技术点 / 技術要素

- C++ 面向对象编程（类的设计与封装）  
- 多文件结构（.h / .cpp 分离）  
- 文件读写（fstream 持久化数据）  
- 简单的控制台交互逻辑  

- C++のオブジェクト指向設計  
- ヘッダーと実装ファイルの分離  
- ファイル入出力（fstream）  
- コンソールベースの操作  

---

## 设计思路 / 設計方針

- 将系统角色拆分为：
  - 员工类（Staff）
  - 经理类（Manager）
  - 管理员类 (Admin)
  - 会议室类（Room）

- 使用文本文件存储预约记录，实现简单持久化  
- 通过菜单驱动控制流程（控制台交互）  
- 不同身份拥有不同操作权限  

- システムの役割をクラスごとに分割  
- テキストファイルによる簡易データ保存  
- メニュー形式による操作  
- 権限に応じた機能制御

## 运行方式（Visual Studio） / 実行方法

1. 使用 Visual Studio 创建空项目  
2. 将所有 .cpp 和 .h 文件添加到项目中  
3. 将所有 .txt 数据文件复制到项目的运行目录  
4. 运行项目  

1. Visual Studioで空のプロジェクトを作成  
2. すべての .cpp と .h ファイルを追加  
3. .txtファイルを実行ディレクトリに配置  
4. 実行  

> 注意：程序依赖 txt 文件存储数据，必须与可执行文件在同一目录，否则无法正常运行  
> ※ txtファイルは同一ディレクトリに配置する必要があります
